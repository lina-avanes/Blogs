Blog 3: Adding users & groups in LINUX

For this week’s blog, I will be demonstrating how to add users and groups in Linux. I will also go over the passwd and shadow files, which contain more information about each user.

Adding users

►In order to add a new user, go to terminal and type:

sudo useradd username

►Sudo will make you the root user and "addusr" is the command which will create the new user. Replace "username" with the name of the new user.

For example:
sudo useradd Ben


Creating groups

►In order to create a new group, we will use the "groupadd" command
►GO to your terminal and type the following commandand make sure to choose your own group name instead of newgroup.

sudo groupadd newgroup

For example:
sudo addgroup Officers


Adding a user to a group

►In order to add a user to a group we will need to follow the command below:

usermod -a -G group-name user-name


►To add an existing user account to a group, use the command "usermod", replacing "group-name" with the name of the group you want to add the user to and "user-name" with the name of the user you want to add.

For example:
sudo usermod -a -G officers Ben

► While a user account can be part of multiple groups, one of the groups is always the “primary group” and the others are “secondary groups”. The user’s login process and files and folders the user creates will be assigned to the primary group.

► Note the -g here. When you use a lowercase g, you assign a primary group. When you use an uppercase -G  , as above, you assign a new secondary group.


shadow and passwd files

►More details about each user an be found in /etc/passwd

ben:x:1000:1000:ben,,,:/home/ben:/bin/bash

►In the informatin above contains the details for user, ben.

  • ben: --> this is the username
  • x: --> this is ben's password (more information about the password can be found in /etc/shadow
  • 1000: --> this is the user ID
  • 1000: --> this is the group ID
  • Ben,,,:: --> this is the user's real/full name.
  • /home/ben: --> this is users home directory
  • /bin/bash --> this is users shell path
