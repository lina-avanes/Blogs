Blog 9: How to SSH to an AWS EC2 Instance

In the previous blog, blog 8, I went over how to create an AWS EC2 Instance. Now in this blog, I will go over how to SSH into a EC2 instance.

Prerequisite

  • Must have an AWS account. (Click here & sign up if you do not have an account)
  • Must have created an AWS EC2 instance and have a key pair. (Click here to learn how)

STEP 1:

Go to your instances page on your AWS account. Select the instance that you would like to SSH into, then copy the IP address.

STEP 2:

Go to your local terminal and locate the key pair file that you downloaded when creating your AWS EC2 instance.

STEP 3:

While in the same path as your key pair file, go ahead and enter the following command:
sudo ssh -i mykey.pem ubuntu@type your EC2 IP here.
Where it says "Type your EC2 IP", you will need the IP from step 1. Then type "yes" for the question "Are you sure you want to continue connecting?"

STEP 4:

You have now been connceted to your AWS EC2 Instance via SSH.
