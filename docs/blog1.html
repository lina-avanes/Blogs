Blog 1: Changing MAC Addresses

Todayâ€™s blog is going to be about Mac Addresses and how to change them via a short python script.

So, what is a MAC address and why would you want to change it?
The MAC address is assigned by the manufacturer of a Network Interface Controller (NIC), and it is stored in the hardware. The NIC is a computer circuit card that allows a computer to connect to a network. To remain anonymous on the internet you should, first of all, change your mac address. Although it may not be the most important thing to do but surely it is one of the steps which you need to do to climb the ladder of anonymity.



Follow the steps below to change your MAC:

STEP 1: Checking the actual MAC

  • Go to your terminal and use the ifconfig command to choose your NIC.

    •        ifconfig

  • As shown in the example below, after typing "ifconfig" in the terminal I can see that my NIC is called "enp0s3" and now I can do "ifconfig enp0s3" and then hit enter to see what my actual MAC is. The MAC is shown in the red box.



    • STEP 2: Basic Python script to change MAC

  • Now type nano mac_changer.py to start your script

    •     nano mac_changer.py

  • Start typing these commands in your mac_changer.py script

  • mac_changer.py will change your MAC to 00:11:22:01:23:45

    •       
      #!/usr/bin/env python
    
      import subprocess
    
      subprocess.call("ifconfig enp0s3 down", shell=True)
    
      subprocess.call("ifconfig enp0s3 hw ether 00:11:22:01:23:45", shell=True)
    
      subprocess.call("ifconfig enp0s3 up", shell=True)

    To learn more about this script click here.



    STEP 3: Run mac_changer.py

  • Run the script by typing "python3 mac_changer.py" in the terminal

    •     python3 mac_changer.py

  • If you get no error after running the script then you have done everything correctly



    • STEP 4: Confirm that your MAC has been changes

  • Use the command "ifconfig" along with your NIC name to see if your MAC has been changed

  • Since my NIC name is "enp0s3" I will type the following command in the terminal 

    •         ifconfig enp0s3

  • As you can see my MAC has been changed to 00:11:22:01:23:45



    • And that is ONE way how you can change your MAC address.
