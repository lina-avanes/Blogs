Blog 1: Changing MAC Addresses

Python script simplified:

  • In the first line add the Python shebang using --> #!/usr/bin/env python

  • The second line of code will import subprocess module The subprocess is need
    to execute bash commands through python --> import subprocess

  • The third line will use the “subprocess.call”. The "call" function just means that it would not call the next command until it is done with the current command. “ifconfig enp0s3 down” will bring down the interface. The “Shell=True” will pass the argument --> subprocess.call("ifconfig enp0s3 down", shell=True)

  • The fourth line of code will change your current MAC address to your desired MAC --> subprocess.call("ifconfig enp0s3 hw ether 00:11:22:01:23:45", shell=True)

  • The last line will turn your interface on and now your MAC has been changed --> subprocess.call("ifconfig enp0s3 up", shell=True) 

    •       
      #!/usr/bin/env python
    
      import subprocess
    
      subprocess.call("ifconfig enp0s3 down", shell=True)
    
      subprocess.call("ifconfig enp0s3 hw ether 00:11:22:01:23:45", shell=True)
    
      subprocess.call("ifconfig enp0s3 up", shell=True)
