Blog 2: File Permissions In Linux

When looking at Linux file permissions, at first, they might seem complicating. However, once learning the meanings of these permissions, they will no longer seem difficult. Lets get started.

Getting started

  • Figure 1 shows what each symbol and letter means in file permissions.

    • "-" --> File
    "d" --> Directory
    "r" --> Read
    "w" --> Wrire
    "x" --> Execute

          Figure 1

  • Now that we know what each letter and symbol means, we can move on and analyze an actual file permission example shown in figure 2

    • -rwxrwxrwx

          Figure 2

  • In order to understand the file permission in figure 2, we need to know that it is consist of 3 parts. Figure 3 separates the three parts and goes over what each part means.

    • -rwx rwx rwx

          Figure 3

  • In figure 3, The first part/group which is "-rwx" belongs to the owners of the file. It means that the owners can read,write, and execute the file.
